# Flutter To-Do App

IS 325: MOBILE APPLICATION DEVELOPMENT - GROUP ASSIGNMENT

This is a simple To-Do List application built with Flutter, featuring a beautiful UI, task management, and local data persistence.

## Features

Add Tasks: Easily add new tasks to your list.
View Tasks: Scrollable list of all your to-do items.
Mark Complete: Check off tasks as you complete them.
Delete Tasks: Remove tasks by swiping or tapping the delete icon.
Persistence: Tasks are saved locally so you don't lose data when closing the app.
Beautiful UI: Modern gradient background and smooth animations.

## How to Run

1.  Clone the repository:
   
    git clone <your-repo-url>
    cd todoapp


2.  Install dependencies:
 
    flutter pub get
  

3.  Run the app:
  
    flutter run
